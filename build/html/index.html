  3. Конфигуратор МПК-Пересвет. Конфигуратор …

МПК-Пересвет - платформа для автоматизации технических объектов, от агрегата до цеха. Также может быть использована для систем типа «умный дом». Возможности: сбор данных, хранение, обработка, выполнение определённых действий на базе происходящих событий. Автоматизация вычислений, бизнес-процессов, происходящих в рамках моделируемого объекта.
