Документация SDK, V0.1
======================

**Документация SDK**/ SDK Documentation
SDK (Software Development Kit) — это набор инструментов создания программного обеспечения для конкретной платформы, включая строительные блоки, отладчики и группу библиотек кода, таких как набор подпрограмм, специфичных для выбранной операционной системы.
Документация SDK содержит инструкции по эффективному использованию SDK.
