Modbus Коннектор V0.1
=====================

**Modbus Коннектор**

Коннектор – это программа для приёма данных из устройств и передачи их в платформу. Связь с платформой реализуется через протокол связи WebSocket. Связь с устройствами может быть реализована с использованием различных протоколов прикладного уровня, например Modbus, OPC, и др.  

Коннектор для протокола Modbus TCP/IP  платформы  написан на языке Python версии 3.10 с использованием библиотеки pyModbus.. Он реализует соединение с устройством по протоколу Modbus и отправку полученных данных   на платформу по протоколу WebSocket.
	Обмен данными устройства с платформой осуществляется в асинхронном режиме за счет использования библиотеки asyncio.
