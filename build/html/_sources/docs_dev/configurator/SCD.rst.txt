Документация по исходному коду программного обеспечения, V0.1
=============================================================

**Исходный код** / Source Code Documentation  (SCD)
Документация по исходному коду содержит все компьютерные программы, относящиеся к программному продукту. Он служит справочником для разработчиков, которые могут работать над более поздними версиями программного обеспечения
