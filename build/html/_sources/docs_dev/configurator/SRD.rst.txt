Дорожная карта продукта, V0.1
=============================

**Дорожная карта продукта** / Roadmap description  (RMD)
Дорожная карта продукта содержит план действий относительно того, как программный продукт будет развиваться с течением времени. Он служит руководством как для бизнеса, так и для технических команд.
